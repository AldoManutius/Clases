Tablas

    •

    Las tablas son para representar contenido tabular en un conjunto de filas y columnas

    Las tablas NO son para maquetar

    Aunque no tegamos etiquetas para definir columnas per se, todas las filas deben tener el mismo número de columnas

    Tabla básica

    El atributo border es estético, solo se lo pongo para aclararme, luego se lo tengo que quitar

    Fila B Dato6 Dato7 Dato8 Dato9
